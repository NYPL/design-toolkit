General tables

Go to top

Use tables sparingly, only for tabular data. Use the basic-table mixin to format a table. All tables should have a thead element with the corresponding headers.

One Two Three Four
some value some value some value some value
some value some value some value some value
… … … … 

.my-table {
  @include basic-table;
}
    

<table class="my-table">
  <thead>
    <tr>
      <th>One</th>
      <th>Two</th>
      <th>Three</th>
      <th>Four</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>some value</td>
      <td>some value</td>
      <td>some value</td>
      <td>some value</td>
    </tr>
    <tr>
      <td>some value</td>

    </tr>
  </tbody>
</table>

Search results summary

Presents a comma-separated list of search terms or filters along with the number of results. The result count has a special span tag (should be number-formatted e.g.: 1,232). Each applied facet or term has a button element to remove the facet and refresh the search results.

NOTE: Avoid adding a space between the facet and the remove button so that it won't wrap to the next line.

1,807 resources found for Young Adult , Fiction . 

.results-summary {
  @include results-summary;
}
    

<div class="results-summary">
  <span class="results-count">1,807</span> resources found for
  <span class="facet">
    <strong>Young Adult<button title="Remove Young Adult filter">×</button></strong>
  </span>,
  <span class="facet">
    <strong>Fiction<button title="Remove Fiction filter">×</button></strong>
  </span>.
</div>

Search results pagination

This widget is presented only in the bottom of the results (pagination will never appear on top). Page count appears in its own span tag.
