Square Button

A very generic squarish button w/ very a very simple inverted hover state. Also a version that utilizes a blue button pattern. 


.my-button-class {
  @include basic-button;
}
.request-button {
  @include request-button;
}
      

<button class="my-button-class">Button</button>
<button class="request-button">Request a Button</button>

Rounded Button

A very simple version of the existent rounded button style.


.my-rounded-button {
  @include basic-button($border-radius: 1.3rem);
}
      

<button class="my-rounded-button">Button</button>
